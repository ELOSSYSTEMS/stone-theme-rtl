# Devkit overview (public preview)

Core idea:
A “component-first” development model for advanced Shopify themes, emphasizing:
- reusable, versionable theme components
- minimal toolchain (avoid heavy bundlers)
- clear interface between reusable components and project-specific composition

Key official resources:
- Devkit repo (public preview): https://github.com/archetype-themes/devkit
- GitHub org: https://github.com/archetype-themes
- Devkit landing page: https://reference-theme.archetypethemes.co/
- Membership discussion (signals private additions): https://github.com/orgs/archetype-themes/discussions/32
