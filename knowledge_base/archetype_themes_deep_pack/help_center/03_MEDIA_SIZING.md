# Media sizing guidance

Source:
- Image and video size recommendations: https://help.archetypethemes.co/en/articles/2716

Key points to encode:
- Use appropriate aspect ratios per section type (hero, product grid, etc.).
- Prefer JPG over PNG for load time (where suitable).
- Define expected dimensions to reduce CLS.

Transferable spec:
- Document section-by-section image aspect ratio expectations.
- Reserve layout space for media.
