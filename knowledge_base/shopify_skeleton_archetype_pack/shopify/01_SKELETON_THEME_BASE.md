# Skeleton Theme baseline

## Key official sources
- Shopify CLI `theme init` — explains Skeleton default if no repo specified:
  https://shopify.dev/docs/api/shopify-cli/theme/theme-init
- "Create a theme" — tutorial that uses Skeleton as a starting point:
  https://shopify.dev/docs/storefronts/themes/getting-started/create
- Skeleton theme repository (structure + conventions):
  https://github.com/Shopify/skeleton-theme
- Theme Store requirements (Skeleton is the only approved base codebase for Theme Store development):
  https://shopify.dev/docs/storefronts/themes/store/requirements

## Skeleton repository highlights (what to study)
- Standard theme folders: assets, blocks, config, layout, locales, sections, snippets, templates
- JSON template scaffolding for editor customization
- Modular asset tags: `{% stylesheet %}` and `{% javascript %}`
- Dedicated critical CSS pattern (`critical.css`)
