# Pixel Union — manual update doctrine

Pixel Union frames manual updating as the standard process: obtain latest version and transfer content/settings/customizations.  
Source: https://support.pixelunion.net/hc/en-us/articles/360038477493-Updating-a-theme-manually citeturn3search1

## Key practice (Verified)
- Open theme editors side-by-side and copy settings section-by-section.  
Source: https://support.pixelunion.net/hc/en-us/articles/360038477493-Updating-a-theme-manually citeturn3search1

## Practical doctrine for STONE (Inference)
- Treat “settings portability” as a design requirement:
  - consistent schema keys
  - stable section names
  - minimal breaking schema changes
